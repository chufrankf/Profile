Personal Restaurant Chooser      GitHub

March 2017 - Present

Early alpha

The purpose of this project is to build a personal restaurant selection based off the outputs of yelp, and google maps. Find restaurants similar to the users preferences and experiences and provide a quick interface to dismiss/block/save/choose a restaurant. If the user wants to avoid restuarants he/she is used to, the user can choose the proportion of new restaurants to select. Then select the options from those. The goal is to allow quick choices of what to eat, while remaining true to the users preferences and history. Also give the user the ability to see more different restaurant types that he/she might like.
